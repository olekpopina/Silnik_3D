Opis szczegółowy

Klasa zarządzająca silnikiem graficznym.

Klasa Engine jest odpowiedzialna za inicjalizację okna, rysowanie obiektów, obsługę wejścia z klawiatury i myszy, a także za renderowanie sceny i kontrolowanie gry.
